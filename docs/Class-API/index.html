Class

Wynncraft Class Resource

These endpoints return general information about the five Wynncraft classes and their corresponding ability trees. Valid class names are archer, assassin, mage, shaman and warrior, in lowercase letters only.

This information only changes when the game updates.

Ability Tree

GET https://web-api.wynncraft.com/v3/ability/tree/{class name}

Detailed description of the archetypes, abilities and the ability tree of a Wynncraft class exactly as shown in the in-game ability tree menu.

Note: To resemble the colors of the in-game menu text, some string values include minecraft color codes, denoted by the character & and the following number or letter.

Ability Tree Object

Field Type Description Options
archetypes Object Object of archetype objects mapped by their name
pages Object Object of page objects that map page numbers of the ability tree to ability objects found on that page

Archetype Object (Ability Tree)

Field Type Description Options
name String Display name of the archetype
description String Long description of the archetype
shortDescription String Short description of the archetype
icon String Website-internal mapping to an icon

Ability Object (Ability Tree)

Field Type Description Options
name String Display name of the ability
archetype String Archetype this ability belongs to null if it does not belong to an archetype
icon String Website-internal mapping to an icon
slot Number Menu slot this ability is shown at in the in-game ability tree menu 0 - 53
coordinates Object Menu slot this ability is shown at in the in-game ability tree menu in a more readable format
description Array of Strings Description text of the ability, separated by lines
requirements Object Requirements needed to unlock this ability
links Array of Strings Other abilities directly connected to this ability null if no links exist
locks Array of Strings Other abilities that get blocked when unlocking this ability null if this ability does not lock other abilities

Coordinates Object (Ability Tree)

Field Type Description Options
x Number x position of the ability or connector 1 - 9
y Number y position of the ability or connector 1 - 6

Requirements Object (Ability Tree)

Field Type Description Options
ABILITY_POINTS Number Ability points required to unlock this ability 1 or 2
NODE String Another ability that has to be unlocked required to unlock this ability
ARCHETYPE Object Certain amount of archetype abilities required to unlock this ability

Archetype Object (Ability Tree Requirements)

Field Type Description Options
name String Name of the required archetype
amount Number Amount of archetype abilities needed

Response

{
    "archetypes": {
        "{name}": {
            "name": String,
            "description": String,
            "shortDescription": String,
            "icon": String
        },
        // ...
    },
    "pages": {
        "1": {
            "{name}": {
                "name": String,
                "archetype": String || null,
                "icon": String,
                "slot": Number,
                "coordinates": {
                    "x": Number,
                    "y": Number
                },
                "description": [String],
                "requirements": {
                    "ABILITY_POINTS": Number,
                    "NODE": String,
                    "ARCHETYPE": {
                        "name": String,
                        "amount": Number
                    }
                },
                "links": [String] || null,
                "locks": [String] || null
            },
            // ...
        },
        "2": {
            // ...
        },
        // ...
    }
}

Ability Tree Map

GET https://web-api.wynncraft.com/v3/ability/map/{class name}

Returns the layout of the ability tree of one of Wynncraft's classes.

Ability Map Object

Field Type Description Options
pages Number Amount of pages in the ability tree
map Array of Objects All items belonging to the map

Item Object (Ability Map)

Field Type Description Options
TYPE String Type of the item ABILITY, CONNECTOR
COORDINATES Object Coordinates of the ability or connector
META Object Meta information about the item
FAMILY Array of Strings Family of the ability if the item is an ability, or families of all the abilities a connector is connecting

Coordinates Object (Ability Map)

Field Type Description Options
X Number x position of the ability or connector 1 - 9
Y Number y position of the ability or connector 1 - 6

Meta Object (Ability Map Item)

Field Type Description Options
ICON String Website-internal mapping to an icon
PAGE String Page number of the ability tree this item belongs to
ID String (Optional) ID of an ability
LINKS Array of Strings (Optional) Directions in which a connector is connecting

Response

{
    "pages": Number,
    "map": [
        {
            "TYPE": String,
            "COORDINATES": {
                "X": Number,
                "Y": Number
            },
            "META": {
                "ICON": String,
                "PAGE": String,
                "ID": String,
                "LINKS": [String]
            },
            "FAMILY": [String]
        },
        // ...
    ]
}

Get

GET https://web-api.wynncraft.com/v3/class/{class name}

General information about a class and its archetypes.

Class Object

Field Type Description Options
id String Name of the class
name String Display name of the class and its reskinned version
lore String Description of the class
overallDifficulty Number Difficulty of playing this class 1 - 3
archetypes Object Object of all the archetypes of this class, mapped by their name

Archetype Object (Class)

Field Type Description Options
name String Display name of the class
difficulty Number Relative difficulty of playing this archetype 1 - 3
damage Number Relative damage of this archetype 1 - 3
defence Number Relative defence of this archetype 1 - 3
range Number Relative range of this archetype 1 - 3
speed Number Relative speed of this archetype 1 - 3

Response

{
    "id": String,
    "name": String,
    "lore": String,
    "overallDifficulty": Number,
    "archetypes": {
        "{name}": {
            "name": String,
            "difficulty": Number,
            "damage": Number,
            "defence": Number,
            "range": Number,
            "speed": Number
        },
        // ...
    }
}

List

GET https://web-api.wynncraft.com/v3/classes

Returns an objects that maps all of Wynncraft's classes to their display name (including their reskinned type) and an estimation of the overall difficulty of playing this class.

Response

{
    "{name}": {
        "name": String,
        "overallDifficulty": Number
    },
    // ...
}
