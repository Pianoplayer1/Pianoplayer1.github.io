News

Wynncraft News Resource

Latest News

GET https://web-api.wynncraft.com/v3/latest-news

A list of the three latest news articles from the Wynncraft Forums.

News Object

Field Type Description Options
title String Title of the forum thread
date String Date of publication in a human-readable format
forumThread String Link to the original forum thread
author String Username of the poster
content String HTML content
comments String Number of comments in the thread

Response

[
    {
        "title": String,
        "date": String,
        "forumThread": String,
        "author": String,
        "content": String,
        "comments": String
    },
    // ...
]

Latest Tweets

GET https://web-api.wynncraft.com/v3/latest-tweets

A list of the five latest tweets from Wynncraft's Twitter account.

Tweet Object

Field Type Description Options
content String The text content of the tweet along with an HTML link to the original tweet
media String Link to the image of the tweet null if the tweet does not contain an imahe
date String Date of publication in a human-readable format
id String ID of the tweet
retweet String Whether the twete is a retweet True, False

Response

[
    {
        "content": String,
        "media": String,
        "date": String,
        "id": String,
        "retweet": String
    },
    // ...
]
