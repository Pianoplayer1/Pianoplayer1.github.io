Introduction

Wynncraft Web API Reference

This unofficial documentation is about the web API used for Wynncraft's own website. That API is technically not meant to be used by external parties, but it has some unique endpoints and other advantages over Wynncraft's regular API that can make it appealing to use. Be aware, though, that the whole web API is always subject to unannounced changes, so be careful when using the routes laid out in this document.

Version

The current and at the moment only version of the web API is Version 3. Thus, all URLs for the API begin with https://web-api.wynncraft.com/v3/. In the top right corner, you can see this version on all pages of this document. Should there ever be more versions, you will be able to select the desired version there.

Advantages

While the API can be less stable than Wynncraft's regular API, it does come with several advantages:

  • Rate limit: The web api currently has no rate limit at all!
  • Connections: Some endpoints provide more statistics at once than their corresponding legacy/v2 counterparts. Examples for this are the forum links in the Player API, the number of online guild members in the Guild API or the ability to search by multiple categories in the Item API.
  • Information: Unique API endpoints like the whole class section provide (mostly) up-to-date information about the game, including for example detailed information about the new ability trees.
