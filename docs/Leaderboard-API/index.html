Leaderboard

Wynncraft Leaderboard Resource

Guild

GET https://web-api.wynncraft.com/v3/leaderboards/guild

returns the same response as

GET https://api.wynncraft.com/public_api.php?action=statsLeaderboard&type=guild&timeframe={timeframe}

Player

GET https://web-api.wynncraft.com/v3/leaderboards/player/{solo|overall}/{type}

returns the same response as

GET https://api.wynncraft.com/v2/leaderboards/player/{solo|overall}/{type}

PvP

GET https://web-api.wynncraft.com/v3/leaderboards/pvp

returns the same response as

GET https://api.wynncraft.com/public_api.php?action=statsLeaderboard&type=pvp&timeframe={timeframe}
