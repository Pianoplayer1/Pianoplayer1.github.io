Item

Wynncraft Item Resource

Data

GET https://web-api.wynncraft.com/v3/item/data/{item name}

Detailed information about an item.

Item Data Object

Field Type Description Options
meta Object General information about the item
identifications Object Possible identifications of the item

Item Meta Object

Field Type Description Options
name String Display name of the item
rarity String The item's rarity Normal, Unique, Rare, Legendary, Fabled, Mythic, Set
type String The item's type Helmet, Chestplate, Leggings, Boots, Ring, Necklace, Bracelet, Dagger, Spear, Wand, Bow
category String Category the item belongs to accessory, armor, weapon
powderSlots Number Available powder slots
dropType String Drop type of the item never, lootchest, NORMAL
quest String Associated quest null if the item is not associated with a quest
level Number Item level
attackSpeed String Attack speed of the item null, SUPER_SLOW. VERY_SLOW, SLOW, NORMAL, FAST, VERY_FAST, SUPER_FAST
lore String Lore of the item null if there is no lore
restrictions String Possible item restrictions null, Quest Item, Untradable
base Object Basic defence or damage
requirements Object Item requirements

Search

Category Searching

GET https://web-api.wynncraft.com/v3/item/category/{category name}

Returns all items present in the category passed in the url, valid categories are: helmet, chestplate, leggings,boots, ring, necklace, bracelet,dagger, spear, wand, bow, beginner, low, medium, experimented, endgame, normal, unique, rare, legendary,fabled, mythic, set.

Multiple categories can be chained using the + character.

Name Searching

GET https://web-api.wynncraft.com/v3/item/search/{query}

Returns a list of Wynncraft items matching the search name. The name is case-insensitive and can be partial.

Item Search Object

Field Type Description Options
name String Display name of the item
rarity String The item's rarity Normal, Unique, Rare, Legendary, Fabled, Mythic, Set
type String The item's type Helmet, Chestplate, Leggings, Boots, Ring, Necklace, Bracelet, Dagger, Spear, Wand, Bow
attackSpeed String Attack speed of the item null, SUPER_SLOW. VERY_SLOW, SLOW, NORMAL, FAST, VERY_FAST, SUPER_FAST
level Number Item level
