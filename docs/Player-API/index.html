Player

Wynncraft Player Resource

Classes

GET https://web-api.wynncraft.com/v3/player/{username}/classes

An object with basic information about all Wynncraft classes of a player, mapped by class name.

Class Overview Object

Field Type Description Options
combat Number Combat level of that class 1 - 106
combatXp Number XP percentage required to get to the next combat level
totalLevels Number The sum of all profession levels and the combat level 1 - 1690
gamemode Object Information about the gamemodes of that class

Response

{
    "{class name}": {
        "combat": Number,
        "combatXp": Number,
        "totalLevels": Number,
        "gamemode": {
            "craftsman": Boolean,
            "hardcore": Boolean,
            "ironman": Boolean,
            "hunted": Boolean
        }
    },
    // ...
}

Class Statistics

GET https://web-api.wynncraft.com/v3/player/{username}/classes/{class name}

Get detailed statistics for a certain class of a player.

Class Object

Field Type Description Options
name String Name of the class
level number The sum of all profession levels and the combat level minus the twelve initial profession levels 1 - 1690
dungeons Object Dungeon statistics for that class
raids Object Raid statistics for that class
quests Object Quest statistics for that class
itemsIdentified Number Amount of items identified on this class (currently broken)
mobsKilled Number Amount of mobs killed on this class
pvp Object PvP statistics for that class
blocksWalked Number Number of blocks walked on this class (currently broken for big numbers)
logins Number Total logins on this class
deaths Number Total deaths on this class
playtime Number Minutes played on this class
gamemode Object Information about the gamemodes of this class
skills Object Information about the skillpoints of this class
professions Object Information about the profession and combat levels of this class
discoveries Number Total discoveries discovered on this class
eventsWon Number Total events won on this class
preEconomyUpdate Boolean Whether this class has reached maximum level before update 1.18
totalLevels Number The sum of all profession levels and the combat level

Class Dungeons Object

Field Type Description Options
completed Number Total completed dungeons on this class
list Object Object that maps each dungeon name to a Dungeon object; uncompleted dungeons are not included in this object

Class Dungeon Object

Field Type Description Options
completed Number (Optional) Total completions of the standard version, if any
completedCorrupted Number (Optional) Total completions of the corrupted version, if any

Class Raids Object

Field Type Description Options
completed Number Total completed raids on this class
list Array of Raid Objects List of all completed raids and their completion numbers

Class Raid Object

Field Type Description Options
name String Name of the raid
completed Number Total completions of the raid

Class Quests Object

Field Type Description Options
completed Number Total quests completed on this class
list Array of Strings Title of each completed quest and miniquest

Class PvP Object

Field Type Description Options
kills Number Total PvP kills on this class
deaths Number Total PvP deaths on this class

Class Gamemode Object

Field Type Description Options
craftsman Boolean Whether the class is in the craftsman gamemode
hardcore Boolean Whether the class is in the hardcore gamemode
ironman Boolean Whether the class is in the ironman gamemode
hunted Boolean Whether the class is in the hunted gamemode

Class Skills Object

Field Type Description Options
strength Number Skillpoints currently assigned to strength
dexterity Number Skillpoints currently assigned to dexterity
intelligence Number Skillpoints currently assigned to intelligence
defence Number Skillpoints currently assigned to defence
defense Number (Duplicate) Skillpoints currently assigned to defence
agility Number Skillpoints currently assigned to agility

Class Professions Object

farming Object Progress of the farming profession on this class
woodcutting Object Progress of the woodcutting profession on this class
fishing Object Progress of the fishing profession on this class

cooking Object Progress of the cooking profession on this class
weaponsmithing Object Progress of the weaponsmithing profession on this class
woodworking Object Progress of the woodworking profession on this class
armouring Object Progress of the armouring profession on this class
combat Object Progress of the combat level on this class

Field Type Description Options
level Number Current level of that profession 1 - 132
xp Number Experience needed to get to the next level of that profession null if the level is at its maximum (132)

List

GET https://web-api.wynncraft.com/v3/players

A list of all online players on Wynncraft and their current servers.

Player List Object

Field Type Description Options
onlinePlayers Number Total number of online players
players Object Maps each online player to their respective server

Response

{
    "onlinePlayers": Number,
    "players": {
        "{username}": String,
        // ...
    }
}

Statistics

GET https://web-api.wynncraft.com/v3/players/{username}

General information about a player, such as overall statistics.

Player Statistics Object

Field Type Description Options
multiples Boolean Unknown
username String Minecraft username of the player
uuid String Minecraft UUID of the player
meta Object Overall player statistics

Player Meta Statistics Object

Field Type Description Options
rank String Full rank of the player null if the player has no rank
shortenedRank String Shortened rank of the player null if the player has no rank
rankColor String || Object Color information of the player's rank String of #000 if the player has no rank
supportRank String Store-bought rank of the player null if the player has no rank
guild Object Information about the player's guild
totalLevels Number Total profession and level of all the player's classes
playtime Number Total playtime in minutes
location Object Information about the current server the player is playing on
firstJoin Date First join on Wynncraft Date in ISO-8601 format with milliseconds
lastJoin Date Latest join on Wynncraft Date in ISO-8601 format with milliseconds
killedMobs Number Total number of mobs killed
forumLink Object Connected forum account
ranking Object Positions on the leaderboards (Currently broken)

Rank Color Object

Field Type Description Options
main String HEX color code of the main rank color
sub String HEX color code of the secondary rank color

Meta Guild Object

Field Type Description Options
name String Name of the guild the player is in null if the player is not in a guild
rank String Guild rank of the player null, RECRUIT, RECRUITER, CAPTAIN, STRATEGIST, CHIEF, OWNER
rankStars String Star prefix of the player in guild chat null if the player is a recruit or not in a guild
meta Object Overall player statistics

Meta Location Object

Field Type Description Options
online Boolean Displays whether the player is currently online
server String The current Wynncraft server the player is playing on null if the player is not online

Meta Forum Link Object

Field Type Description Options
username String (Optional) Forum username of the player
ign String Minecraft username of the player
id Number (Optional) Forum ID of the player

Response

        },
        "firstJoin": Date,
        "lastJoin": Date,
        "killedMobs": Number,
        "forumLink": {
            "username": String,
            "ign": String,
            "id": Number
        },
        "ranking": {
            "guild": null,
            "player": {
                "solo": {
                    "combat": null,
