Guild

Wynncraft Guild Resource

Statistics

GET https://web-api.wynncraft.com/v3/guild/{guild name}

General guild information, including level, members, territories, xp and more.

Guild Object

Field Type Description Options
name String Name of the guild
prefix String The guild's prefix or tag
members Objects Object of member objects, sorted by guild rank
xp Number Percentage of guild XP required to get to the next guild level
level Number Guild level
created Date Date that this guild was created at ISO-8601 Date with milliseconds
territories Number The amount of territories the guild currently owns
banner Object (Optional) The banner of the guild
onlineMembers Number The number of guild members currently online
totalMembers Number The number of total players in the guild

Guild Member Object

Field Type Description Options
location Object Current online status and Wynncraft server of this member
uuid String The Minecraft UUID of the player
joined Date Date that this member joined the guild at ISO-8601 Date with milliseconds
contributed Number Amount of XP contributed to the guild
guildRank Number Position in the guild ranked by join date (oldest date first)

Guild Banner Object

Field Type Description Options
base String Base colour of the banner WHITE, ORANGE, MAGENTA, LIGHT_BLUE, YELLOW, LIME, PINK, GRAY, LIGHT_GRAY, CYAN, PURPLE, BLUE, BROWN, GREEN, RED, BLACK
tier Number Tier of the banner's base
structure String Current banner structure
layers Array of Objects List of layers on this banner

Guild Banner Layer Object

Field Type Description Options
colour String The colour of this layer WHITE, ORANGE, MAGENTA, LIGHT_BLUE, YELLOW, LIME, PINK, GRAY, LIGHT_GRAY, CYAN, PURPLE, BLUE, BROWN, GREEN, RED, BLACK
pattern String The pattern used by this layer CIRCLE_MIDDLE, SQUARE_BOTTOM_LEFT, SQUARE_BOTTOM_RIGHT, SQUARE_TOP_LEFT, SQUARE_TOP_RIGHT, HALF_HORIZONTAL, STRIPE_BOTTOM, STRIPE_TOP, HALF_VERTICAL, STRIPE_LEFT, STRIPE_CENTER, STRIPE_RIGHT, STRIPE_MIDDLE, STRAIGHT_CROSS, STRIPE_DOWNLEFT, STRIPE_DOWNRIGHT, CROSS, DIAGONAL_LEFT, DIAGONAL_UP_RIGHT, TRIANGLE_TOP, TRIANGLE_BOTTOM, RHOMBUS_MIDDLE, TRIANGLES_TOP, TRIANGLES_BOTTOM, CURLY_BORDER, BORDER, STRIPE_SMALL, BRICKS, GRADIENT, CREEPER, SKULL, FLOWER, MOJANG, DIAGONAL_LEFT_MIRROR, DIAGONAL_RIGHT, GRADIENT_UP, HALF_HORIZONTAL_MIRROR, or HALF_VERTICAL_MIRROR

Location Object

Field Type Description Options
online Boolean Displays whether the player is currently online
server String The current Wynncraft server the player is playing on null if the player is not online
